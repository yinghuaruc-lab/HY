# 体重管理健康实验室

> 一个帮助用户通过28天科学实验建立健康饮食习惯的轻量级Web应用。

## 产品简介

体重管理健康实验室是一款基于"习惯养成"理念的健康管理工具。用户通过28天的实验周期，每天记录体重、感受和协议完成情况，获得数据驱动的个性化建议。

## 核心功能

### 1. 8项健康协议

实验包含8个经过验证的健康管理协议，用户每天可多选完成：

| 协议 | 说明 |
|------|------|
| 含糖饮料清零 | 用无糖饮品替代奶茶、果汁、碳酸饮料等 |
| 晚饭后不吃零食 | 晚餐结束后不再进食直到次日早餐 |
| 每日步数≥8000 | 通过步行或运动达到8000步以上 |
| 蛋白质早餐 | 早餐以高蛋白食物为主，减少碳水 |
| 晚饭提前 | 晚餐时间提前至睡前3小时以上 |
| 盘子法 | 每餐用小一号盘子，蔬菜占一半 |
| 酒精限制 | 减少或戒除酒精摄入 |
| 零食替换 | 用健康零食（坚果、水果）替代高热量零食 |

### 2. 每日记录

每天填写以下数据：

- **晨起体重** (kg)
- **饥饿感评分** (1-5)
- **暴食冲动评分** (1-5)
- **协议完成勾选** (多选)

### 3. 进度追踪

- 28天日历视图
- 每日完成率显示
- 可回看和修改历史记录

### 4. 数据分析报告

28天结束后自动生成：

- 起始/结束体重对比
- 体重变化趋势
- 各协议完成率排行
- 个性化结论与建议

## 技术特点

- **无需登录** - 数据保存在本地存储 (localStorage)
- **轻量级** - 单HTML文件，无需后端
- **响应式** - 适配桌面和移动设备
- **流畅动画** - 元素淡入效果，界面精致
- **隐私安全** - 数据仅存储在用户本地

## 使用流程

```
开始实验 → 每日记录(28天) → 查看报告 → 重新开始
```

## 设计理念

1. **简化选择** - 不再让用户选择单个协议，一次性提供全部8项
2. **渐进习惯** - 通过多选机制，让用户自然找到适合自己的协议
3. **数据驱动** - 用实际数据说话，避免主观臆断
4. **温和激励** - 结论和建议采用正向、鼓励的语气
